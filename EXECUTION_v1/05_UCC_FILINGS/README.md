## üîÑ CONTINUATION & AMENDMENTS

### UCC-1 Continuation

**Requirement:** UCC-1 filings expire after **5 years**

**Action:** File UCC-3 Continuation Statement within 6 months before expiration

**Reminder:** Set calendar reminder for **April 2031** (if filing in 2026)

---

### UCC-3 Amendment

**When to File:**
- Collateral description changes
- Secured party name changes
- Debtor name changes
- Termination (upon facility payoff)

**Action:** File UCC-3 Amendment within 10 days of change
