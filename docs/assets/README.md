# OPTKAS Brand Assets

Place the following image files in this directory. The HTML pages reference these exact filenames.

## Required Files

| Filename | Description | Used On |
|:---------|:------------|:--------|
| `optkas-seal-gold.png` | Imperia seal with OPTKAS text, lions, eagle (white/gold bg) | Hero section — both pages |
| `optkas-seal-dark.png` | Imperia seal WITHOUT OPTKAS text (dark bg) | Footer area — both pages |
| `optkas-logo-wings.png` | Horizontal metallic wings logo (chrome/gold) | Navigation bar — both pages |
| `optkas-logo-ornate.png` | Ornate gold text with filigree underneath | CTA section — both pages |
| `optkas-logo-metallic.png` | Horizontal metallic banner logo | Footer — both pages |
| `optkas-sword-seal.png` | MMXXGI sword ornate diamond badge | Bond section header |
| `sovereign-banner.png` | "ALL SYSTEMS LIVE" dashboard banner | Hero section — both pages |
| `xrpl-badge.png` | 877570XRPL lightning badge (gold) | XRPL section + CTA phone banners |

## Notes
- All images should be PNG with transparent backgrounds where applicable
- Images use CSS `drop-shadow` and `filter` effects for glow
- `onerror` fallbacks hide missing images gracefully — text-only fallback works
