# GitHub Pages Setup Instructions

## Enable GitHub Pages for OPTKAS1 Funding Portal

### Step 1: Access Repository Settings
1. Go to [github.com/unykornai/TC](https://github.com/unykornai/TC)
2. Click on **Settings** tab
3. Scroll down to **Pages** section

### Step 2: Configure Source
1. Under **Source**, select **Deploy from a branch**
2. Under **Branch**, select **main** branch
3. Under **Folder**, select **/docs** folder
4. Click **Save**

### Step 3: Access the Portal
After 2-3 minutes, your site will be available at:
**https://unykornai.github.io/TC/**
